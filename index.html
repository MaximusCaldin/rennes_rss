Ici Rennes Off'

Bienvenue sur le flux RSS non officiel de Ici Rennes, le dispositif de communication de Rennes Métropole.

FLUX

https://ici-rennes-off.skiserv.eu/feed.php

Le flux aggrège les 20 derniers articles de Ici Rennes, au format RSS sans délai par rapport à la publication sur l'application officielle. Pour plus d'informations techniques et le processus de création de cet outil je vous invite à aller voir l'article dédié sur mon blog Flux Ici Rennes. Le code source est disponible sur Github MaximusCaldin/rennes_rss.

Pour s'abonner au flux, une application est nécessaire. Voici par exemple la procédure pour ajouter un flux dans Thunderbird, mais il existe aussi des applications smartphone ou des extensions navigateur web.
